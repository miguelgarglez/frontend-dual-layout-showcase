# Plan

Transformar esta prueba técnica en un case study de portfolio con identidad propia, narrativa orientada a producto y copy consistente en todo el flujo. El enfoque será iterativo: primero identidad visual base, luego copywriting, y finalmente pulido UX/accesibilidad.

## Scope
- In:
  - Rebranding visual (nombre, identidad, tokens, iconografía, imágenes, screenshots)
  - Reescritura de textos UI (desktop y mobile-first)
  - Storytelling de proyecto para recruiter/hiring manager
- Out:
  - Integración backend real o persistencia
  - Reescritura completa de arquitectura de componentes
